- 标题：智御 · OS 智能运维 Agent（OpsGuard Agent）
- 副标题：用安全护栏架构驾驭 AI 运维的不确定性
- 队伍/参赛信息

## 幻灯片 2：痛点与挑战
- 传统运维：门槛高、风险大、不可溯源
- AI 运维的核心矛盾：LLM 推理不确定 → 可能生成高危/越权/被注入的命令
- 一句话：难点不是「让 AI 会运维」，而是「让 AI 运维不出事」

## 幻灯片 3：产品一页纸
- 自然语言入口 + MCP 打通 OS + 安全护栏约束不确定性

## 幻灯片 4：总体架构图
- B/S 架构，后端同时提供 REST + MCP + 静态控制台

## 幻灯片 5：安全护栏七阶段闭环
- 接收 → 抗注入 → 感知 → 推理 → 校验 → 执行 → 分析
- 强调：每一步都可溯源
